instalación

para gente normal (archivo .exe)

  1. descarga el .zip más reciente desde lanzamientos
  2. descomprime y guarda el .exe junto con las carpetas assets y lang en la misma carpeta.
  3. haz doble clic en main.exe (o bert.exe)

para frikis de tropecientos de iq

git clone https://github.com/berrota/super-bert-bros
cd super-bert-bros
pip install -r requirements.txt
python main.py

requisitos: Python 3, Git, y las dependencias de requirements.txt.
