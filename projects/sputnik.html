instalación

para gente normal:

  1. ir a lanzamientos
  2. seleccionar la versión más reciente
  3. descargar el archivo .jar
  4. moverlo a tu directorio de mods (probablemente %appdata%/.minecraft/mods)

requisitos: Minecraft 1.21.11 y Fabric 0.17.3

para frikis de 200 de IQ:

git clone https://github.com/retuci0/sputnik
cd sputnik
gradlew build # Windows
# o
chmod +x gradlew && ./gradlew build # Linux/Mac

El archivo .jar resultante estará en /build/libs/sputnik-X.X.jar.

es posible que haya problemas al usarlo junto a otros mods, especialmente junto a otros clientes, pues pueden conflictar entre ellos.

interfaz

acción método
activar / desactivar módulos clic izq.
abrir ajustes de módulo clic dcho.
Mover marcos clic izq. + arrastrar
expandir / contraer marco clic dcho. en la cabeza del marco
cerrar marco clic dcho. en la cabeza del marco
modificar ajustes clic izq.
restablecer ajustes shift + clic dcho.
ver descripción pasar cursor sobre módulo / ajuste

protips:


  • shift mientras pasas el puntero sobre ajustes de modo para ver los modos disponibles
  • clic dcho. para ciclar hacia atrás en ajustes de modo
  • rueda del ratón para mover la interfaz verticalmente
  • shift + rueda del ratón (presionar) para mover todos los marcos a un rango visible
